# casa_v2

Projeto em C++ de computação gráfica que renderiza uma casa usando OpenGL 3.3 Core Profile, organizado com CMake.

## Requisitos

- Linux
- C++23 compatível
- CMake ≥ 3.0.0
- Make
- OpenGL 3.3 Core Profile

## Como compilar

```bash
mkdir build
cd build
cmake ..
make
```

## Como executar

```bash
./casa_v2
```

Dependências já inclusas. Código testado em Linux.
