# Computacao_Grafica
Repositório destinado para códigos de computação gráfica. 

## <h2 id=3>Requisitos</h2>

- Linux
- C++23 compatível
- CMake ≥ 3.0.0
- Make
- OpenGL 3.3 Core Profile

## Como compilar

```bash
mkdir build
cd build
cmake ..
make
```

## Como executar

```bash
./engine
```

## Edição
Para executar as modificações feitas no código será preciso [compila-lo](3) novamente.
